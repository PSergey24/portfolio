Balance Parser

Task: We have a large database of information about companies that has been downloaded from official sources. The main task is to extract data on the economic performance of companies. The tables are divided into active and passive part. Each part can contain up to over 200 different labels. Labels are conventionally divided into levels. The sum of the child labels must be equal to the parent label. Tables may contain errors in data and typos in labels. The main difficulty lies in the fact that there is no single structure. Each company can prepare reports in any format.

Tools:

- Python 3.7

- Docker-compose

- RabbitMQ

- AWS

- Postgresql

- Asyncio

Achievements:

- A parser/extractor was developed that recognizes 96-98% of cases

- A system of microservices has been created that parses pages, extracts information and saves it to the database

- Some research work
