Employees Parser

Task: We have a large database of information about companies that has been downloaded from official sources. The main task is to extract data about employee's count of companies.

The main difficulty is that there is no single format. Each company specifies this information in any format. You can find many examples with tables, these can be tables of varying complexity. In addition to tables, information can be placed in the text using numbers and phrases. Information about employees can be divided into different groups (full time employees, part time employees, interns, directors, season employees, by profession, etc.). Information can be duplicated 2-4 times, while the values ​​may differ from each other. Pages can have different sizes (from 1 to 20 screens).

Tools:

- Python 3.7

- Spacy

- Docker-compose

- RabbitMQ

- AWS

- Postgresql

- Asyncio

Achievements:

- Information extractors were developed based on Spacy library and pattern recognition. Both system were combined for increasing quality of recognition. Overall recognition accuracy over 95%

- A system of microservices has been created that parses pages, extracts information and saves it to the database

- Some research work

- Experience in managing international students for the purpose of data labeling. In english language
